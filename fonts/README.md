# About /fonts

#### Simple Sans

Default webfont for First Data

See repo for all typefaces (eot, woff, ttf included for cross device/browser compatibility).

#### Icon Fonts

##### Reference
[CSS](https://github.com/dfuzr/fd-ignite_payments/blob/master/app/styles/less/fonts.less)

##### Usage

```
<i class="icon icon-close"></i>
```

Size of icon controlled by `font-size` property, not `width` / `height`

```
.icon-close {
  font-size: 24px;
}
```


Raw [JSON](https://github.com/dfuzr/fd-ignite_payments/tree/master/app/fonts/fd-icons.json) file for updating/appending to set.
